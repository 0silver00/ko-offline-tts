========================================
  TTS Generator - 오프라인 한국어 음성 생성기
========================================

■ 실행 방법
──────────────────────────────────────
  1. "TTS Generator.bat" 파일을 더블클릭합니다.
  2. 브라우저가 자동으로 열리며 http://localhost:8080 에 접속됩니다.
  3. 텍스트를 입력하고 "Generate" 버튼을 클릭하면 음성이 생성됩니다.
  4. 생성된 음성을 브라우저에서 재생하거나 MP3로 다운로드할 수 있습니다.

■ 사전 요구 사항
──────────────────────────────────────
  ★ Python 설치가 필요하지 않습니다!
  python/ 폴더에 Python 3.14 Embeddable이 내장되어 있어
  별도의 설치 과정 없이 바로 실행됩니다.

■ 브라우저 접속 URL
──────────────────────────────────────
  http://localhost:8080

■ 인터넷 연결
──────────────────────────────────────
  인터넷 연결이 필요하지 않습니다.
  모든 처리가 브라우저 내부에서 오프라인으로 동작합니다.
  USB 복사만으로 다른 PC에서도 즉시 실행 가능합니다.

■ 포함된 구성 요소
──────────────────────────────────────
  - Python 3.14 Embeddable (python/ 폴더, 로컬 서버용)
  - sherpa-onnx WASM (브라우저 내장 TTS 엔진)
  - ko_KO-kss_low 한국어 음성 합성 모델 (models/ 폴더)
  - lamejs (브라우저 내장 MP3 인코더)

■ 바로가기 만들기
──────────────────────────────────────
  1. "TTS Generator.bat" 우클릭 → 바로 가기 만들기
  2. 생성된 바로가기 우클릭 → 속성 → 아이콘 변경
  3. tts-icon.ico 파일 경로를 지정하면 마이크 아이콘이 표시됩니다.

■ 배포 시 포함해야 하는 폴더/파일
──────────────────────────────────────
  offline-korean-tts/
  ├── TTS Generator.bat    ← 실행 파일
  ├── start.bat            ← 실행 파일 (동일)
  ├── server.py            ← 로컬 서버

  └── models/              ← 한국어 모델 (필수)
