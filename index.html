📖 사용 방법
1
텍스트 입력

텍스트 입력 영역에 한국어 텍스트를 직접 입력하거나, 📂 버튼으로 .txt 파일을 업로드하세요. 최대 10,000자까지 지원합니다.

2
설정 조정

음성 속도(0.5x~2.0x)를 슬라이더로 조절하세요. 기본값은 속도 1.0x 입니다.

3
음성 생성

Generate 버튼을 클릭하면 브라우저 내부에서 음성이 합성됩니다. 인터넷 연결이 필요하지 않습니다.

4
미리듣기 및 다운로드

합성 완료 후 오디오 플레이어로 바로 재생하고, 파일명을 입력한 뒤 MP3 다운로드 버튼으로 저장하세요.
